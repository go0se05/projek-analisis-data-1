## Installation
1. Clone repository:
```
git clone https://github.com/yourusername/projek-analisis-data-1.git
cd projek-analisis-data-1
```
2. Install Dependencies
```
pip install -r requirements.txt
```

## Run App
1. Notebook

Google colab-notebook .
```

```
cd dashboard
